PixelSort Visualizer

HTML5 Canvas Vanilla JS

Overview

PixelSort Visualizer animates bubble sort, merge sort, and quicksort on the same random array so it’s easy to compare how they actually behave, not just their Big-O on a whiteboard.

Everything is drawn directly to a <canvas> element on every frame; there’s no DOM per bar, which is most of why it stays smooth even at a few hundred elements.

Role & Contributions

  • Solo project, built for a data structures course
  • Implemented all three sorting algorithms as step-generators so the animation can pause/resume
  • Wrote the Canvas rendering loop and the simple playback controls
