StudyGroup Finder

Python Flask React PostgreSQL

Overview

StudyGroup Finder matches students in the same course into small study groups based on availability, built with two teammates for a class project.

It was my first time building real authentication instead of faking it, which turned out to be most of the actual work.

Role & Contributions

  • Team of three; I owned the backend, authentication, and database schema
  • Built the matching algorithm that groups students by overlapping free time
  • Reviewed teammates’ pull requests for the React front end
